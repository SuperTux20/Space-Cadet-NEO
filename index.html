Space Cadet NEO

Welcome to my long term programming project.

Space Cadet NEO is ia faithful reconstruction of "3D Pinball: Space Cadet", which came out in 1995 and was packaged with Mictosoft PLUS! for Windows 95. The game was eventually included automatically with Windows ME, 2000, and XP. However, a 64-bit version was never made for Vista, because there was a nasty collision error that made the game unplayable, and so it was largely forgotten by Microsoft. I am here to fix that.
